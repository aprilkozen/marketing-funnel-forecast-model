# Marketing Funnel Forecast Model

A full-funnel B2B marketing forecast model that projects MQLs and SQLs
by channel and quarter, then flows them through to modeled pipeline and
new ACV.

## What it does

- **Channel-level forecasting** across Web/organic, Nurture, Paid, and
  Conferences, split by segment (Mid-Market, Enterprise)
- **Seasonality-weighted phasing** — monthly targets are driven by each
  month's share of quarterly web-session volume, so the forecast tracks
  real demand patterns instead of dividing evenly by three
- **Goal-vs-actual gap analysis** by channel, surfacing where to
  reallocate spend
- **Dashboard** rolling MQL and SQL forecasts into modeled Opps,
  Closed-Won, and New ACV via conversion assumptions

## How to read it

- **Blue cells** are inputs — change them to run scenarios
- **Black cells** are formulas
- **Green cells** link across sheets
- **Highlighted cells** are the key assumptions driving output

## Note

Figures are synthetic and illustrative — this is a demonstration of
forecasting and dashboard methodology, not any company's actual data.
